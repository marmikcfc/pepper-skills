# Referral Program Designer

Design a referral program that converts happy customers into your best acquisition channel.

## When to Use
- "Design a referral program"
- "Turn customers into advocates"
- "Build a word-of-mouth growth engine"
- "Launch a give-X / get-X program"

## Prerequisites
- `ANTHROPIC_API_KEY`
- Product pricing model, customer LTV and CAC

## Workflow

**Step 1: Define program economics**

| Metric | Value |
|--------|-------|
| ACV or MRR | |
| Customer LTV | |
| Current CAC | |
| Max referral reward (% of LTV) | |

**Step 2: Choose the incentive model**

| Model | Best for |
|-------|---------|
| Dual-sided (both get reward) | Consumer / PLG |
| Referrer-only (credit/cash) | B2B SaaS |
| Upgrade (free month / feature) | Subscription products |
| Cash | High-ACV products |

**Rules of thumb:**
- Reward should feel meaningful, not transactional
- SaaS: 1-3 months credit is common
- Cash: 5-15% of first payment is typical
- Higher reward isn't always better — test amounts

**Step 3: Design the mechanics**

| Element | Decision |
|---------|----------|
| Trigger (when to ask?) | After first value / 30 days / renewal |
| Sharing method | Unique link, code, or both |
| Reward delivery | Instant or on convert |
| Attribution window | 30 / 60 / 90 days |
| Fraud prevention | Limit per account, same-domain rule |

**Step 4: Write program copy**

```
Write copy for a referral program for [product]:
Incentive: [referrer reward] + [referee reward]
Audience: [who we're asking]
Tone: [casual/professional]

Write:
1. Program headline
2. How it works (3 steps max)
3. Referral invitation email (150 words, customer to friend)
4. In-app prompt (< 50 words)
5. Confirmation email when referral converts
```

**Step 5: Launch plan**

| Phase | Activity |
|-------|----------|
| Pre-launch | Build share page, set up tracking, test flow |
| Soft launch | Email top 20% most engaged customers |
| Full launch | Email all active customers |
| Amplification | In-app prompt, onboarding touch, renewal ask |
| Optimization | A/B test incentive and copy after 30 days |

## Output
Referral program design with economics, incentive model, mechanics, copy suite, and launch plan.
